# Taskline Application

A task management web application built with Node.js and Express.

## Description

Taskline is a web-based task management tool that helps users organize and track their tasks efficiently.

## Features

- Task creation and management
- Clean, user-friendly interface
- Custom font support for better readability

## Technologies Used

- **Backend:** Node.js, Express
- **Frontend:** HTML, CSS, JavaScript
- **Other:** CORS for cross-origin requests

## Installation

1. Clone the repository:
```bash
   git clone https://github.com/TJRiley-1/TaskLine_Webapp.git
```

2. Navigate to the project directory:
```bash
   cd TaskLine_Webapp
```

3. Install dependencies:
```bash
   npm install
```

4. Start the server:
```bash
   node server.js
```

5. Open your browser and visit:
```
   http://localhost:3000
```
